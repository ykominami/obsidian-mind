第12章 マイクロフラストレーション（小さなイライラ）の解消
1.「大きな課題」より「小さな不便」に注目せよ
2.なぜ「小さな不便」がねらい目なのか？
(1)多くの人が共感する
(2)解決がインプル
(3)即効性がある
(4)マネタイズしやすい
「面倒をお金で解決できるなら払う」という人は多い。
3.具体例：YouTubeでの小さなイライラ
・タイムスタンプ入力が面倒
YouTube APIで動画時間を取得し、自動で目次を作成
・動画内容を思い出せない
ChatGPT APIで動画の要約を生成
・勉強動画でメモが取りづらい
視聴中にEnterキーを押すと、タイムスタンプ付きでメモが保存される　あぷり

6.マイクロフラストレーションを見つける方法
1.毎日の作業を振り返る
2.「面倒くさい」と思った瞬間を記録する
3.他人の口コミやレビューを読む
4.APIで解決できるかを調べる

7.マイクロフラストレーション解消アプリの強み
