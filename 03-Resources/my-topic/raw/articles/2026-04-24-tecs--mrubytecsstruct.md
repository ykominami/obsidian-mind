# <no title>

TECS::StructTAG

C 言語の構造体を扱うためのクラスであり、signature に現れる構造体に合わせて MrubyBridgePlugin により生成されます。
TAG の部分は、構造体のタグ名に置き換えます。

== スーパークラス ==

Object

== クラスメソッド ==

new()

== メソッド ==

MEMBER

構造体メンバの値を読み出します。
MEMBER はメンバ名に置き換えます。

MEMBER = val

構造体メンバの値を書き込みます。
MEMBER はメンバ名に置き換えます。
