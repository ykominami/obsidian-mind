# 技術文書化

## Definition

仕様書・マニュアル・設計ドキュメントを正確かつ再利用可能な形式で作成・維持する能力。コードベースやドキュメントからの知識抽出、LLMツールを活用した大規模ドキュメント生成、読者の視点に立った構造設計を含む。

## Proficiency Levels

| Level | Description |
|-------|-------------|
| beginner | 既存フォーマットに従ってドキュメントを更新できる |
| intermediate | ゼロからドキュメント構造を設計し、ツールを活用して大規模生成できる |
| advanced | ドキュメント生成パイプラインを設計・自動化できる。仕様駆動開発を主導できる |
| expert | 組織のドキュメント標準を策定し、他者の技術文書化を指導できる |
