# ナレッジマネジメント

## Definition

個人・チームの知識を構造化・検索可能な形式で管理し、将来の自分や他者が再利用できる状態に保つ能力。情報の分類・ルーティング、グラフ型ナレッジベースの設計、LLMを活用した大規模ナレッジ整理を含む。

## Proficiency Levels

| Level | Description |
|-------|-------------|
| beginner | ノートを取り、フォルダで管理できる |
| intermediate | タグ・リンクを活用した検索可能なナレッジベースを維持できる |
| advanced | 大規模なナレッジ整理をツールで自動化できる。graph-firstな構造設計ができる |
| expert | 組織レベルのナレッジシステムを設計・導入できる |
