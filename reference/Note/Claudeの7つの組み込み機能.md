# Claudeの7つの組み込み機能

> [godofprompt](https://x.com/godofprompt) のX投稿（2025-05-22）より

## 1. プロジェクト — コンテキストの永続化

プロジェクトはブランドガイドライン・過去の成果物など全体コンテキストを保持する。一度ドロップするだけで、プロジェクト内の新チャットはすべて完全なコンテキストから始まる。

## 2. カスタムスタイル — 出力品質の最大化

任意のプロンプト調整より大きく出力品質を変える。「懐疑的なベテランオペレーター」のように、弱い論理に反論するスタイルが有効。設定 > スタイル > カスタムスタイルを作成。

## 3. メモリ + 過去チャット検索の連携

メモリで会話間の好み・プロジェクト・コンテキストを保持。過去チャット検索で特定の決定内容を引き出せる。設定で両方オンにする。

## 4. モデル選択 — Sonnet vs Opus

- **Sonnet 4.6**: 日常業務に最適。Opus 4.7より高速・低コスト
- **Opus 4.7**: 複雑な推論・長時間コードセッション・深い分析向け
- デフォルトはSonnet。必要なときだけOpusに切り替える。

## 5. モバイル音声モード — 思考ツールとして

20分歩きながら問題を声に出して考え、Claudeに要約させる。最高の思考はデスクでは起こらない。

## 6. CLAUDE.md — プロンプトより効果的

50〜80行のプロジェクトコンテキスト（技術スタック・規約・ファイル構造）を一度書く。Claude Codeが毎セッション自動的に読み込む。

## 7. アーティファクトAPI — アプリビルダーとして

アーティファクト内でClaudeのAPIを呼び出せる。クライアントブリーフ生成ツール・クイズエンジン・コンテンツ評価システムなどを構築可能（"Claudeception"）。
