# 第6章　運用・モニタリング
## Agent Temasの並列起動に価値があるか
### 並列が有効
- タスクが真に独立している
- 各エージェントが別の成果物を生成
- 時間短縮がトークンコストを正当化する
### 並列が無駄
- タスクに直列依存関係がある
- 1エージェントで逐次処理できる
- 単純な作業・単発タスク
## Webダッシュボードで使用量を定期的にチェックする
使用量の感覚を養うことが節約の出発点
-　日時別　機能どれぐらい使ったか
- タスク別　どのタスクが高いか
- 週次レビュー　傾向を把握
## 5時間制限を逆用してウィンドウを確保する
