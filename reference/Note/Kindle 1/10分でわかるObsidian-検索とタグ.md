# 10分でわかるObsidian — 検索とタグ

## 第4章 検索・タグ

### 検索フィールド

| フィールド | 用途 |
|-----------|------|
| `file:` | ファイル名で検索 |
| `path:` | パスで検索 |
| `tag:` | タグで検索 |
| `section:` | セクション見出しで検索 |
| `task:` | タスクアイテムで検索 |

検索クエリはブックマーク保存可能。

### タグ戦略

1. **目的を明確にして一貫性のあるルールを定める**
   - 種類: `#議事録`, `#アイデア`, `#資料`
   - テーマ: `#マーケティング`, `#開発`
   - 状態: `#状態/進行中`, `#状態/完了`

2. **階層タグで粒度と柔軟性を両立**（`/` で階層化）

3. **リンクとタグの使い分け**
   - リンク → ノート間の直接的な関連性・相互参照
   - タグ → ノートの属性・分類（テーマ、状態、種類）

### 検索結果のダッシュボード化

````markdown
```query
tag:#タスク tag:#状態/進行中 line:(今日)
```
````

## 第5章 デイリーノート

デイリーノートを日々の活動ハブにする構成例:

```
## 今日の目標
### タスク
### アイデア
### 振り返り
[プロジェクト名]
```

MOCノート（Map of Contents）で全体を俯瞰する。
