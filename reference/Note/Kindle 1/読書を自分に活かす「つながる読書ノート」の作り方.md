# 読書を自分に活かす「つながる読書ノート」の作り方

著者: [[小田やかた]] / Kindle版

## コアメソッド

- 印象に残った文章を**抜き書き**する
- 興味を引いたキーワードやフレーズを**リンク化**する
- 抜き書きの下に「そのとき考えたこと・感じたこと・要約」を追記
  - 自分の考えは別ノートに切り出す
- 左側縦線 = 本からの引用部分
- 縦線の下 = 自分の考え・感想

## 読書ノートの構成要素

| 要素 | 内容 |
|------|------|
| **レビュー** | 2〜3行の簡単な感想 |
| **Action** | 読後に行動したいこと（意識・考え含む） |
| **学び** | 新たに得た内容・再認識・役立ちそうな内容・自分の常識と異なる考え |
| **引用** | Kindleはハイライトから取り込み／紙は手書き・写真・テキスト化 |

## つながりを作る工夫

- 印象に残ったフレーズは**逃がさずリンク**する
- タグでつなげる — 日本十進分類法ベース:
  - `0 総記` / `1 哲学・宗教` / `2 歴史・地理` / `3 社会科学`
  - `4 自然科学` / `5 技術・家庭・工業` / `6 産業`
  - `7 芸術・体育` / `8 言語` / `9 文学`
  - カスタム例: `101_情報科学・IT` / `102_哲学・心理学・倫理学・宗教`
- 「ジャンル」×「テーマ」で分類・整理
- メモは**別ノートにしてつなげる**
- aliases を活用してノートをつながりやすくする
- 関連ノートは必ず見返す
