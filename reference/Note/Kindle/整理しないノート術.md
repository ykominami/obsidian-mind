## 保存しているノートの種類
- 読書ノート
	- 読んだ本の感想、印象に残った点、生美になった内容、引用文など
	- AmazonのページをObsidian Web Clipperで取り込む
	- Kindle版の書籍のハイライトの取り込み
	- 読書ノートを一覧にしたページを自動生成
- 着想メモ
	- 日々の生活の中でふと浮かんだアイデアや発信内容のネタ
- 発信物
	- ブログ記事、電子書籍など発信したコンテンツの原稿
	- Xの投稿文の下書き
- 知識メモ
	- トピックごとの再利用可能な断片
- 思考メモ
	- 考え事や心のもやもや、言語化できそうでできない思考の断片を吐き出すノート
