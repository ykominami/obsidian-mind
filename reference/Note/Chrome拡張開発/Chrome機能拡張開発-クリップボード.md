画面上でマウスカーソルで選択状態にした文字列を、クリップボードにコピーするGoogle Chromeの拡張機能の作り方を教えて

はい、承知いたしました。マウスで選択したテキストを自動的にクリップボードにコピーするGoogle Chrome拡張機能の作り方を、初心者の方にも分かりやすくステップバイステップで解説します。

---

### はじめに

ここで作成する拡張機能は、ウェブページ上でテキストをマウスでドラッグして選択し、マウスのボタンを離した瞬間に、その選択したテキストを自動でクリップボードにコピーするというものです。

作業は非常にシンプルで、**2つのファイルを作成**し、それをChromeに読み込ませるだけで完了します。

---

### ステップ1：作業フォルダの作成

まず、拡張機能のファイルを保存するためのフォルダをPC内の好きな場所（デスクトップなど）に作成します。フォルダ名は「CopyToClipboard」など、分かりやすい名前を付けておきましょう。

---

### ステップ2：マニフェストファイルの作成 (`manifest.json`)

拡張機能の設計図となる`manifest.json`というファイルを作成します。このファイルには、拡張機能の名前、バージョン、必要な権限などの情報を記述します。

1. テキストエディタ（メモ帳やVisual Studio Codeなど）を開きます。
    
2. 以下のコードをコピーして貼り付けます。
    
3. 作成した「CopyToClipboard」フォルダ内に、`manifest.json`という名前で保存します。
    

JSON

```
{
  "manifest_version": 3,
  "name": "選択してコピー",
  "version": "1.0",
  "description": "マウスで選択したテキストを自動的にクリップボードにコピーします。",
  "permissions": [
    "clipboardWrite"
  ],
  "content_scripts": [
    {
      "matches": [
        "<all_urls>"
      ],
      "js": [
        "content.js"
      ]
    }
  ],
  "icons": {
    "16": "icon16.png",
    "48": "icon48.png",
    "128": "icon128.png"
  }
}
```

#### **コードの簡単な解説**

- `"manifest_version": 3`: 最新のChrome拡張機能の仕様バージョンを指定します。
    
- `"name"`: 拡張機能の名前です。
    
- `"version"`: 拡張機能のバージョンです。
    
- `"description"`: 拡張機能の簡単な説明です。
    
- `"permissions": ["clipboardWrite"]`: クリップボードへの書き込みを許可するための重要な設定です。
    
- `"content_scripts"`: ウェブページ上で直接動作するスクリプトを指定します。
    
    - `"matches": ["<all_urls>"]`: すべてのウェブサイトでこの拡張機能を有効にします。
        
    - `"js": ["content.js"]`: 実行するJavaScriptファイルの名前を指定します。
        

---

### ステップ3：メインの処理を記述する (`content.js`)

次に、実際のコピー処理を行うJavaScriptファイルを作成します。

1. テキストエディタで新しいファイルを開きます。
    
2. 以下のコードをコピーして貼り付けます。
    
3. 「CopyToClipboard」フォルダ内に、`content.js`という名前で保存します。
    

JavaScript

```
document.addEventListener('mouseup', () => {
  // マウスで選択されているテキストを取得
  const selectedText = window.getSelection().toString().trim();

  // 選択されたテキストが存在する場合のみ処理を実行
  if (selectedText.length > 0) {
    // クリップボードにテキストを書き込む
    navigator.clipboard.writeText(selectedText)
      .then(() => {
        // 成功した場合（任意でコンソールにメッセージ表示）
        console.log(`「${selectedText}」をコピーしました。`);
      })
      .catch(err => {
        // 失敗した場合
        console.error('クリップボードへのコピーに失敗しました: ', err);
      });
  }
});
```

#### **コードの簡単な解説**

- `document.addEventListener('mouseup', ...)`: ウェブページ上でマウスのボタンが離されたとき（`mouseup`イベント）に、指定した処理を実行します。
    
- `window.getSelection().toString()`: ユーザーがマウスで選択しているテキストを取得します。`.trim()`で前後の余分な空白を削除しています。
    
- `if (selectedText.length > 0)`: テキストが実際に選択されている場合のみ、コピー処理に進むようにしています。
    
- `navigator.clipboard.writeText(selectedText)`: 取得したテキストをクリップボードに書き込みます。これがコピー処理の核となる部分です。
    

---

### ステップ4：アイコンの準備（任意）

拡張機能にアイコンを設定すると、管理画面で見つけやすくなります。これは任意ですが、設定しておくことをお勧めします。

1. 好きな画像（PNG形式）を用意し、以下の3つのサイズにリサイズします。
    
    - `16x16` ピクセル
        
    - `48x48` ピクセル
        
    - `128x128` ピクセル
        
2. それぞれを `icon16.png`, `icon48.png`, `icon128.png` という名前で、「CopyToClipboard」フォルダに保存します。
    

_もしアイコンがなくても拡張機能は動作します。その場合、`manifest.json`から`"icons"`の部分を削除しても構いません。_

---

### ステップ5：Chromeに拡張機能をインストール

作成したファイルをChromeに読み込ませます。

1. Google Chromeを開き、アドレスバーに `chrome://extensions` と入力してEnterキーを押します。
    
2. 拡張機能の管理ページが開きます。右上の「**デベロッパー モード**」のトグルスイッチをオンにします。
    
3. 「**パッケージ化されていない拡張機能を読み込む**」というボタンが表示されるので、クリックします。
    
4. ファイル選択ダイアログが開くので、先ほど作成した「**CopyToClipboard**」フォルダを選択し、「選択」ボタンをクリックします。
    
5. 一覧に「選択してコピー」という拡張機能が追加されれば成功です！
    

これですべての作業は完了です。今すぐ herhangiのウェブページを開き、テキストをマウスで選択してみてください。マウスのボタンを離した瞬間に、テキストがクリップボードにコピーされているはずです。
