● セルタイプとは何か（tecs wiki 版）

  セルタイプ（celltype） はコンポーネントの型定義です。セルはいずれかのセルタイプに属します。セルタイプがコンポーネント
  の「構造」と「振る舞い」を規定し、そのインスタンスがセルです。

  CDL でのセルタイプ定義

  [celltype-specifier] celltype tMyType {
      call   sSignature cCallPort;    // 呼び口（機能を要求）
      entry  sSignature eEntryPort;   // 受け口（機能を提供）
      attr {
          int32_t myAttr = 10;        // 属性（ROM配置、読出し専用）
      };
      var {
          int32_t myVar;              // 内部変数（RAM配置、読み書き可）
      };
  };

  セルタイプが持つ要素

  ┌───────────────────────┬──────┬──────────┬─────────────────────────────────────────────────┐
  │         要素          │ 配置 │ 書き換え │                      説明                       │
  ├───────────────────────┼──────┼──────────┼─────────────────────────────────────────────────┤
  │ 受け口（Entry Port）  │ —    │ —        │ 機能を提供する口。allocator / inline 指定子あり │
  ├───────────────────────┼──────┼──────────┼─────────────────────────────────────────────────┤
  │ 呼び口（Call Port）   │ —    │ —        │ 機能を要求する口。optional / omit 指定子あり    │
  ├───────────────────────┼──────┼──────────┼─────────────────────────────────────────────────┤
  │ 属性（Attribute）     │ ROM  │ 不可     │ セル記述で初期値を上書き可能な定数              │
  ├───────────────────────┼──────┼──────────┼─────────────────────────────────────────────────┤
  │ 変数（Variable）      │ RAM  │ 可       │ セルの実行時内部状態                            │
  ├───────────────────────┼──────┼──────────┼─────────────────────────────────────────────────┤
  │ ファクトリ（Factory） │ —    │ —        │ コンフィギュレーションファイル生成用            │
  └───────────────────────┴──────┴──────────┴─────────────────────────────────────────────────┘

  セルタイプ指定子

  ┌───────────────────────┬──────────────────────────────────────────────────────────────────────────────────────┐
  │        指定子         │                                         意味                                         │
  ├───────────────────────┼──────────────────────────────────────────────────────────────────────────────────────┤
  │ singleton             │ システム中にセルが1つだけ存在するセルタイプ。受け口関数から CELLIDX idx 引数が消える │
  ├───────────────────────┼──────────────────────────────────────────────────────────────────────────────────────┤
  │ active                │ タスク・ハンドラ等、受け口関数を呼ばれなくても動作する                               │
  ├───────────────────────┼──────────────────────────────────────────────────────────────────────────────────────┤
  │ idx_is_id             │ セルの実行時識別に整数値を使用（有効性検査あり）                                     │
  ├───────────────────────┼──────────────────────────────────────────────────────────────────────────────────────┤
  │ generate(Plugin, ...) │ セルタイププラグインを適用                                                           │
  └───────────────────────┴──────────────────────────────────────────────────────────────────────────────────────┘

  C 言語実装との対応

  セルタイプの振る舞いは セルタイプコード（tCelltype.c）に実装します。tecsgen が自動生成する tCelltype_tecsgen.h
  を必須インクルードします。

  セルタイプが提供するマクロ（p_cellcb 定義済みを前提とする短縮形）：

  ┌────────────────────────┬────────────────────────────────┐
  │         マクロ         │              用途              │
  ├────────────────────────┼────────────────────────────────┤
  │ ATTR_属性名            │ ROM 属性へのアクセス           │
  ├────────────────────────┼────────────────────────────────┤
  │ VAR_変数名             │ RAM 内部変数へのアクセス       │
  ├────────────────────────┼────────────────────────────────┤
  │ cCall_func(...)        │ 呼び口関数の呼び出し           │
  ├────────────────────────┼────────────────────────────────┤
  │ FOREACH_CELL(p_cellcb) │ セルタイプ内の全セルを反復処理 │
  ├────────────────────────┼────────────────────────────────┤
  │ NCP_cCall              │ 呼び口配列のサイズ取得         │
  └────────────────────────┴────────────────────────────────┘

  singleton vs 非 singleton の実装差

  // 非シングルトン：idx で個々のセルを識別
  ER eEntry_func(CELLIDX idx, int32_t inval) {
      CELLCB *p_cellcb;
      if (VALID_IDX(idx)) { p_cellcb = GET_CELLCB(idx); }
      else { return E_ID; }
      ...
  }

  // シングルトン：idx 引数なし、CB 取得不要
  ER eEntry_func(int32_t inval) { ... }
