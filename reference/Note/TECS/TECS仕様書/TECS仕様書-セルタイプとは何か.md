● セルタイプとは何か（tecs-0 wiki 版 — 仕様書原文ベース）

  セルタイプ（celltype）
  はコンポーネントの型定義です。セルの構造（口・属性・変数）と振る舞い（セルタイプコード）を規定します。

  CDL でのセルタイプ定義（BNF より）

  celltype tMyType {
      call   sSignature  cCallPort;     // 呼び口
      entry  sSignature  eEntryPort;    // 受け口
      attr { int32_t myAttr = 0; };     // 属性（ROM配置）
      var  { int32_t myVar; };          // 変数（RAM配置）
      require sSig;                      // リクワイア（型整合確認）
      factory { write(...); };           // コンフィギュレーション生成
  };

  セルタイプ指定子

  ┌───────────┬──────────────────────────────────────────────────┐
  │  指定子   │                       意味                       │
  ├───────────┼──────────────────────────────────────────────────┤
  │ singleton │ システム中にセルが1つだけ存在                    │
  ├───────────┼──────────────────────────────────────────────────┤
  │ idx_is_id │ セルの実行時識別に整数値を使用（有効性検査あり） │
  ├───────────┼──────────────────────────────────────────────────┤
  │ active    │ タスク等、CPU 資源割付けを伴うアクティブセル     │
  └───────────┴──────────────────────────────────────────────────┘

  セルタイプが持つ要素

  ┌───────────────────────┬─────────────┬──────────────────────────────────────────┐
  │         要素          │    配置     │                   説明                   │
  ├───────────────────────┼─────────────┼──────────────────────────────────────────┤
  │ 呼び口                │ —           │ 他のセルの機能を要求する口               │
  ├───────────────────────┼─────────────┼──────────────────────────────────────────┤
  │ 受け口                │ —           │ 機能を提供する口                         │
  ├───────────────────────┼─────────────┼──────────────────────────────────────────┤
  │ 属性（attribute）     │ ROM（INIB） │ 読出し専用。セル記述で初期値を上書き可能 │
  ├───────────────────────┼─────────────┼──────────────────────────────────────────┤
  │ 変数（var）           │ RAM（CB）   │ 実行時読み書き可能な内部状態             │
  ├───────────────────────┼─────────────┼──────────────────────────────────────────┤
  │ リクワイア（require） │ —           │ シグニチャの型整合チェック               │
  ├───────────────────────┼─────────────┼──────────────────────────────────────────┤
  │ ファクトリ（factory） │ —           │ コンフィギュレーションファイル生成用     │
  └───────────────────────┴─────────────┴──────────────────────────────────────────┘

  C 言語への写像（tecsgen が生成）

  ┌────────────────────────────┬──────┬──────────────────────────────────────────────┐
  │           構造体           │ 配置 │                     内容                     │
  ├────────────────────────────┼──────┼──────────────────────────────────────────────┤
  │ INIB（Cell INItial Block） │ ROM  │ 呼び口結合先ディスクリプタへのポインタ・属性 │
  ├────────────────────────────┼──────┼──────────────────────────────────────────────┤
  │ CB（Cell Block）           │ RAM  │ var 変数・INIB へのポインタ                  │
  └────────────────────────────┴──────┴──────────────────────────────────────────────┘

  4パターンの組み合わせ:

  ┌───────────┬─────────────────────────────────┬─────────────────────────┐
  │ パターン  │              条件               │          CB 型          │
  ├───────────┼─────────────────────────────────┼─────────────────────────┤
  │ CB + INIB │ var あり & attribute あり       │ tCt_CB { _inib; var; }  │
  ├───────────┼─────────────────────────────────┼─────────────────────────┤
  │ CB のみ   │ attribute なし・呼び口なし      │ var のみ                │
  ├───────────┼─────────────────────────────────┼─────────────────────────┤
  │ INIB のみ │ var なし                        │ typedef tag_CB tag_INIB │
  ├───────────┼─────────────────────────────────┼─────────────────────────┤
  │ 両方なし  │ var も attribute も呼び口もなし │ GET_CELLCB(idx) → 0     │
  └───────────┴─────────────────────────────────┴─────────────────────────┘

  アクセスマクロ（セルタイプコード内）

  // 短縮形（p_cellcb が定義済みを前提）
  ATTR_myAttr       // ROM 属性へのアクセス
  VAR_myVar         // RAM 変数へのアクセス
  cCall_func(...)   // 呼び口関数の呼び出し

  // 通常形（他のセルからアクセス）
  tMyType_ATTR_myAttr(p_that)
  tMyType_SET_myVar(p_that, val)

  呼び口最適化（tecsgen が自動適用）

  ┌──────────────────────────┬──────────────────────────────────────┬────────────────────────────────────────────────┐
  │          最適化          │                 条件                 │                      効果                      │
  ├──────────────────────────┼──────────────────────────────────────┼────────────────────────────────────────────────┤
  │ 単一セル最適化           │ 全結合先が同じセルの同じ受け口       │ 呼び口情報を CB/INIB                           │
  │                          │                                      │ に置かず関数マクロに埋込み                     │
  ├──────────────────────────┼──────────────────────────────────────┼────────────────────────────────────────────────┤
  │ 関数テーブル不要         │ 全受け口が同じセルタイプの同じ受け口 │ VMT を介さず直接呼出し                         │
  ├──────────────────────────┼──────────────────────────────────────┼────────────────────────────────────────────────┤
  │ 受け口ディスクリプタ不要 │ 上記 + 受け口が配列でない            │ 受け口ディスクリプタ → IDX に置換              │
  └──────────────────────────┴──────────────────────────────────────┴────────────────────────────────────────────────┘
