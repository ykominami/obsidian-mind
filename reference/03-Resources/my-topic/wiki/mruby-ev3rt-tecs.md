# mruby on ev3rt+tecs

mruby on ev3rt+tecsは、LEGO Mindstorms EV3向けのTECS統合mruby実装です。

## 提供クラス

| クラス | 説明 |
|-------|------|
| Battery | バッテリ管理 |
| Button | ボタン入力 |
| LED | LEDライト |
| Speaker | スピーカ |
| LCD | ディスプレイ |
| Motor | モータ制御 |
| RTOS | RTOSサービス |
| Color | カラーセンサ |
| Gyro | ジャイロセンサ |
| Ultrasonic | 超音波センサ |
| Touch | タッチセンサ |
| SharedMemory | 共有変数 |
| Balancer | 倒立振子バランサ |
