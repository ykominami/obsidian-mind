## 11. 用語集

| 用語 | 定義 |
|------|------|
| TCD | TECS Component Diagram。TECS コンポーネント構成を視覚的に表現する図 |
| CDL | Component Description Language。TECS のコンポーネント記述言語 |
| セル | コンポーネントのインスタンス |
| セルタイプ | コンポーネントの型定義 |
| 呼び口（Call Port） | 機能を要求するポート |
| 受け口（Entry Port） | 機能を提供するポート |
| シグニチャ | 呼び口・受け口のインターフェース定義 |
| 結合（Join） | 呼び口と受け口を接続する関係 |
